# Pyla Language for VS Code

Syntax highlighting, bracket matching, auto-indent, and comment toggling
(Ctrl+/) for `.pyla` files.

## Install (local)

Copy this folder into your VS Code extensions directory and restart VS Code:

```
# Windows
%USERPROFILE%\.vscode\extensions\pyla-lang.pyla-0.3.0

# macOS / Linux
~/.vscode/extensions/pyla-lang.pyla-0.3.0
```

Or package it properly with `npx @vscode/vsce package` and install the
resulting `.vsix` via "Extensions: Install from VSIX".
